# FundTracker

FundTracker is a simple project designed to track and manage funds in an organized manner. It allows users to record financial data and monitor fund-related information in a structured format.

## Purpose

The purpose of FundTracker is to provide a basic system for maintaining fund records and enabling easy tracking of financial activities.

## Scope

This project focuses on:
- Storing fund-related data  
- Tracking basic transactions  
- Providing a clear overview of available funds  

## Status

The project is under development and may be extended with additional features in the future.
